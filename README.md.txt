# Star Project

This is a machine learning project that classifies stars based on their physical attributes like temperature, luminosity, and radius.

## Dataset

The dataset used is `star-dataset.csv`, containing features such as:

- Temperature (K)
- Luminosity (L/Lo)
- Radius (R/Ro)
- Absolute Magnitude
- Star Type
